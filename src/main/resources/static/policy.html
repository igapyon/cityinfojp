自治体など組織による都市情報ファクトを淡々と集積

cityinfojp のデータポリシー

cityinfojp では自治体などの組織による都市情報のファクトを淡々と集積します。

以下のポリシーでデータを収集します。

  • ファクトにのみ着眼し、解説や論評といった情報は扱いません。
  • その組織の一次情報源に近いファクトを優先して収集します。
  • cityinfojp はデータレビューを重視します。

集められたデータは CC BY 4.0 ライセンスのもと取り扱います。


  • cityinfojp で収集したデータは CC BY 4.0 ライセンスのもと提供します。
  • src/main/resources/static/input ディレクトリおよびそのサブディレクトリに含まれる JSON ファイルが CC BY 4.0 ライセンスの適用対象です。
  • データ提供およびデータ再利用には 表示 4.0 国際 (CC BY 4.0) ライセンスの理解および同意が前提条件です。
  • データ提供で貢献する人は歓迎されます。
  • 提供を希望するデータは github のプルリクエストを経由して貢献することができます。

以下の順でデータを貢献できます。

  • データ生成で JSON データファイルを生成します。

  • データ投入 にデータをプルリクエストします。
