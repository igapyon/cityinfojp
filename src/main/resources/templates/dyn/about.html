cityinfojp の目的

cityinfojp は自治体などの組織による都市情報のファクトを淡々と集積する Webサイトです。
ファクトにのみ着眼し解説や論評といった情報は扱いません。 ファクトは地方自治体など一次情報源に近いレベルを優先して収集します。

cityinfojp の当面の目的

当面は日本の行政(県・市・町・村)からの外出自粛要請や学校の休校情報などの情報を収集して表示します。

オープンソース (OSS)

cityinfojp はオープンソースです。実現するソースコードのすべてが github 上で公開されます。 オープンソース (OSS) ライセンスの理解と同意により、開発への参加・再利用・改変などが可能です。

サイトのステータス

現在このサイトはベータ版のフェーズにあります。一緒に作ってくれる オープンソース活動に理解のある人を募集しています。

開発言語など

cityinfojp は Java, Spring Boot, Thymeleaf, Bootstrap, JSON により構成されます。 実用的な Web アプリであり、有用なサンプルでもあります。 Thymeleaf の静的 HTML ジェネレータとして実装されています。
