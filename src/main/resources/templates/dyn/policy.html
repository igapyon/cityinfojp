cityinfojp のデータポリシー

cityinfojp では自治体などの組織による都市情報のファクトを淡々と集積します。

以下のポリシーとします。

  • ファクトにのみ着眼し解説や論評といった情報は扱いません。
  • その組織の一次情報源に近いファクトを優先して収集します。
  • cityinfojp のデータレビューを通過したデータのみ収集します。

集められたデータはオープンデータとして扱います。

以下のポリシーで取り扱います。

  • オープンソースのルールを理解していることが参加条件です (OSS よりなクリエイティブコモンズを採択予定)。
  • データ提供の貢献者は歓迎されます。
  • 提供を希望をデータは github のプルリクエストを経由して貢献してください。

以下の順でデータを貢献できます。

  • データ生成で JSON データファイルを生成します。
