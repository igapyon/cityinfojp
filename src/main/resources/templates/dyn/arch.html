開発言語

cityinfojp は プログラミング言語Java を利用して開発しています。

ビルドシステム

cityinfojp は Maven をビルドシステムとして利用しています。

ビルドおよび実行方法

mvn test
mvn spring-boot:run

詳しくは pom.xml を参照ください。

フレームワーク

cityinfojp は Spring Boot をフレームワークとして利用しています。

HTML レンダリング

cityinfojp は Thymeleaf を HTML レンダリングエンジンとして利用しています。 ビルドプロセスのなかで Thymeleaf テンプレートから 静的 HTML へコンバートしています。

Webフロントエンド

cityinfojp は Bootstrap を フロントエンド・コンポーネント・ライブラリとして利用しています。

データ形式

cityinfojp は JSON をデータフォーマットとして原則利用しています。 JSON ファイルの読み書きには Jackson ライブラリを利用しています。

開発作業場所

cityinfojp は GitHub を開発拠点としています。

Web ホスト

cityinfojp は Heroku ホスティングサービスを利用しています。
